# 웹 성능 최적화

  • 웹 성능을 끌어올리기 위해 공부한 이론들과 시도한 내용들 정리
  • 랜더링 성능 개선하기 → 화면을 더 빠르게 그리기
  • 로딩 성능 개선하기 -> 각 리소스들을 빠르게 다운로드하기, 지연 시간을 줄이기, 다운로드 양을 줄이기
